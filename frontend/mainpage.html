Search2Emoji

Implementation of a search engine
( Tsinghua Web IR project )

Available functionality:
1. Emoji search
search for custom emojis using Word2Vec model or for default Unicode emojis with normal search

2. Emoji prediction
predict emoji at the end of the sentence via fine-tuned or base BERT models

3. Text editor
default features of a text editor, e.g., font style, font size, save, dark mode
